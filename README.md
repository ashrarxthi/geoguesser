# 🌍 GeoGuesser

**How well do you know the World?**  
Test your geography knowledge here → [Play Now](https://ashrarxthi.github.io/geoguesser/geoguesser.html)

---

## Game Modes

**⌨️ Fill the Map** — Type any country name and watch it light up on the map. 15 minutes on the clock. How many can you name?

**🔍 Name That Country** — A country pulses on the map. Type its name before you run out of guesses. Use hints if you're stuck.

**📍 Find It** — A country name is shown. Click the right spot on the map with no highlights. 10 strikes.

**📚 Learn by Region** — Study countries region by region with capitals, populations, and geography facts. Then quiz yourself!

Built with D3.js + TopoJSON.
